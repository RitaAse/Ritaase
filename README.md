# FeedLens  
### AI-Powered Assessement Summarizer Tool for Educators

FeedLens is an NLP-driven summarization system that transforms raw student feedback into structured, actionable insights.  

It helps educators identify patterns in strengths and weaknesses across an entire class — reducing cognitive overload and saving valuable grading time.

---

## 🚩 Problem

Teachers often:
- Spend hours reviewing large volumes of written feedback
- Struggle to identify patterns across multiple students
- Experience cognitive overload when synthesizing insights manually
- Lack collective performance overviews across assignments

Traditional grading tools focus on quantitative insights — not contextual insight extraction.

---

## 💡 Solution

FeedLens enables teachers to:

1. Upload structured feedback files (Word/Excel/pdf)
2. Automatically analyze textual comments using NLP
3. Detect common strengths and weaknesses
4. Generate contextual summaries
5. Review and refine (if needed) AI-generated insights before publishing (Human-in-the-Loop)

---

## ⚙️ How It Works
File Upload → Text Preprocessing → Feature Extraction → Pattern Detection → Summary Generation → Teacher Review


### NLP Pipeline Includes:
- Text cleaning & normalization
- TF-IDF vectorization
- Topic modeling
- Frequency analysis
- Extractive summarization
- Abstractive-style synthesis
- Optional anonymization of student names

## 📊 Example Output

### Raw Feedback Input
> "Good structure but lacks critical analysis."  
> "Strong argumentation, needs clearer references."  
> "Well-written introduction, weak conclusion."

### Synthesized Output

**Common Strengths**
- Clear structure
- Strong argumentation
- Well-written introductions

**Common Weaknesses**
- Limited critical analysis
- Inconsistent referencing
- Weak conclusions
